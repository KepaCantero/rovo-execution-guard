## Configuration Details

### Parser

| Setting     | Value                     | Purpose                       |
| ----------- | ------------------------- | ----------------------------- |
| parser      | @typescript-eslint/parser | TypeScript AST parsing        |
| ecmaVersion | 2022                      | Modern JavaScript syntax      |
| sourceType  | module                    | ESM modules                   |
| jsx         | true                      | React JSX support             |
| project     | ./tsconfig.json           | Type-aware rules via tsconfig |

### Extends (order matters)

| Preset                                | Purpose                                                   |
| ------------------------------------- | --------------------------------------------------------- |
| eslint:recommended                    | Base JS best practices                                    |
| plugin:@typescript-eslint/recommended | TS-specific rules                                         |
| plugin:@typescript-eslint/strict      | Strict TS rules                                           |
| plugin:react/recommended              | React best practices                                      |
| plugin:react-hooks/recommended        | Hooks rules                                               |
| prettier                              | Disables all formatting rules that conflict with Prettier |

### Key Rules

| Rule                                             | Level | Config                 | Rationale                                  |
| ------------------------------------------------ | ----- | ---------------------- | ------------------------------------------ |
| @typescript-eslint/no-explicit-any               | error | —                      | [GUARDRAIL-1002] Zero any — no exceptions  |
| @typescript-eslint/no-unused-vars                | error | argsIgnorePattern: ^\_ | Dead code detection with underscore escape |
| complexity                                       | error | max: 10                | Prevent overly complex functions           |
| no-console                                       | warn  | —                      | Use structured logging instead             |
| @typescript-eslint/explicit-function-return-type | warn  | allowExpressions: true | Encourage explicit returns                 |

### Settings

| Setting                    | Value         | Purpose                         |
| -------------------------- | ------------- | ------------------------------- |
| react.version              | detect        | Auto-detect React version       |
| import/resolver.typescript | tsconfig.json | Resolve path aliases in imports |

### Prettier Integration (`.prettierrc`)

| Option        | Value  | Purpose                    |
| ------------- | ------ | -------------------------- |
| singleQuote   | true   | Consistent quote style     |
| trailingComma | all    | Cleaner diffs              |
| printWidth    | 100    | Readable line length       |
| tabWidth      | 2      | Consistent indentation     |
| semi          | true   | Explicit semicolons        |
| arrowParens   | always | Consistent arrow functions |

### Ignore Files

| File            | Entries                      |
| --------------- | ---------------------------- |
| .eslintignore   | node_modules, coverage       |
| .prettierignore | node_modules, coverage, .git |
